1 <h2>Instructions</h2>
2 <p>
3  <code>batterystat</code> is a commandline tool to print battery status information.
4 </p>
5 <p>
6  The following options are available:
7 </p>
8 <dl>
9  <dt>-a</dt>
10  <dd>Connect to remote battery address.</dd>
11  <dt>-c</dt>
12  <dd>Prints the count of locally installed batteries and exits.</dd>
13  <dt>-m</dt>
14  <dd>Prints additional battery hardware information like manufacturer, manufacture date etc.</dd>
15  <dt>-n #</dt>
16  <dd>Request information for battery at position #. This is only meaningful for multiple battery systems like Pismo laptops. Default is 1.</dd>
17  <dt>-p</dt>
18  <dd>
19  Return the value of a single property. Allowed property names are: charge, amperage,
20  capacity, maxCapacity, designCapacity, voltage, cycleCount,
21  plugged, charging, manufacturer, manufactureDate, serial, deviceName,
22  timeToEmpty, timeToFullCharge. If not used, all properties are returned.
23  </dd>
24  <dt>-t</dt>
25  <dd>Include time information.</dd>
26  <dt>-v</dt>
27  <dd>Prints version information and exits.</dd>
28  <dt>-x</dt>
29  <dd>Prints output in XML format.</dd>
30 </dl>
