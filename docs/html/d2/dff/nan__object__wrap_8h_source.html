96  /* Unref() marks an object as detached from the event loop. This is its
97  * default state. When an object with a "weak" reference changes from
98  * attached to detached state it will be freed. Be careful not to access
99  * the object after making this call as it might be gone!
100  * (A "weak reference" means an object that only has a
